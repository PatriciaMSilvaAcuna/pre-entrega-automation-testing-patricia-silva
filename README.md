# Proyecto QA Automation

Este proyecto contiene pruebas automatizadas realizadas con Python, Selenium y Pytest para validar funcionalidades del sitio SauceDemo.

## Funcionalidades automatizadas

- Login exitoso
- Validación de inventario
- Verificación de catálogo de productos
- Agregado de productos al carrito
- Validación de carrito de compras

## Tecnologías utilizadas

- Python
- Selenium WebDriver
- Pytest
- WebDriver Manager
- Pytest HTML
- Visual Studio Code

## Instalación de dependencias

Instalar las dependencias necesarias ejecutando:

```bash
pip install -r requirements.txt
```
